## 📌 Overview

ATS Resume Scanner is an intelligent web application designed to analyze resumes against job descriptions using AI and ATS (Applicant Tracking System) concepts. It helps users improve resume quality, identify missing keywords, and increase chances of getting shortlisted.

This project provides detailed resume analysis, ATS score evaluation, skill matching, and visualization insights in a modern and user-friendly interface.

---

## ✨ Features

✅ Upload Resume (PDF/DOCX)  
✅ ATS Score Calculation  
✅ Job Description Matching  
✅ Keyword Analysis  
✅ Resume Skill Detection  
✅ Data Visualization & Charts  
✅ AI-Based Resume Suggestions  
✅ Clean & Responsive UI  
✅ Fast Resume Parsing

## 🚀 Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/kumarsaurav11er-lgtm/Ats-Resume-Scanner.git
```

### 2️⃣ Move into Project Folder

```bash
cd Ats-Resume-Scanner
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run Application

```bash
python app.py
```

### 5️⃣ Open in Browser

```bash
http://127.0.0.1:5000
```
